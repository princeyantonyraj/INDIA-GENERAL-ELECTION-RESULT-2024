📊 Constitution Election 2024 – Power BI Dashboard

This project presents an interactive Power BI dashboard analyzing election-related data for the Constitution Election 2024.

🔹 Project Overview:

The dashboard provides insights into voting patterns, candidate performance, and regional analysis using dynamic visualizations.

🔹 Key Features:
Interactive visuals for state-wise / region-wise analysis
KPI metrics to track total votes, winners, and margins
Filters and slicers for drill-down insights
Clear representation using bar charts, maps, and tables
🔹 Tools & Technologies:

Power BI Data Visualization Data Cleaning Dashboard Design

🔹 Insights Generated:
Identified top-performing candidates/parties
Analyzed voting trends across regions
Compared vote share and winning margins
